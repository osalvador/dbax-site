dbax is a wonderful MVC web Framework made for Oracle PL/SQL

for high-performing database-driven Web applications. Learn More

PL/SQL MVC Framework

DBAX is the first PL/SQL MVC Web framework. Take control of your application code (front-end and back-end).

Web console

Configure your application, review logs, create users... through an intuitive Web console.

Secure

Authenticate your users using LDAP, Simple authentication, RBAC for Authorization... Exploit database features like Virtual Private Database and Real Application Security.

Excepcional performance

Engineered with performance tuning in mind.

dbax is ready for Enterprise

Scale with the Oracle Database. Oracle RAC, Oracle Cloud and the In-Memory option.

Web standards

Friendly URLs, HTML5, CSS3 and Responsive Design, JavaScript and their libraries (AngularJS, ReactJS, whatever).

Fast development

Live changes, scaffolding, user session state, simple url routing, code generators (tapiGen2).

Open Source

DABX is an Open Source project. Not increase the cost of your Oracle database. Compatible with Free Oracle database Express Edition (XE).

Logging and tracing

Automatically record events and data about your application execution. Use it to identify software problems or live monitor.

Easy skill transition for existing Forms developers

Oracle Forms is no longer a strategic development tool. Upskilling and encourages your PL/SQL developers to a real web environment
